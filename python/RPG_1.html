RPG - Tekst baseret rollespil

Et tekstbaseret rollespil er et computerspil, hvor handlingen beskrives gennem tekst frem for grafik.
Spilleren navigerer i en fiktiv verden ved at læse beskrivelser og give kommandoer som f.eks. "gå nordpå" eller "tal med vagten".

Interactionen sker typisk via en kommandolinje.

Spillets kerne ligger i historiefortælling, problemløsning og udvikling af din rolle (karakter) gennem valg og udforskning.

Input fra brugeren

print(f"Hej {name}!")

Man skal huske at bruge input() funktionen med en prompt, og at gemme brugerens input i en variabel.
Og at konvertere input til den rigtige datatype, hvis det er nødvendigt. For eksempel:

Validering af input

Det er vigtigt at validere input fra brugeren for at undgå fejl og sikre, at programmet fungerer som det skal.
For eksempel kan man tjekke om en alder er et tal og ikke en tekst.
Det kan gøres ved at bruge en try-except blok:

try:
    age = int(input("Hvor gammel er du? "))
except ValueError:
    print("Du skal skrive et tal.")

vi skal også huske at tekst input kan være med både store og små bogstaver.

Derfor kan vi bruge .lower() eller .upper() metoderne på inputtet.
så vi kan sammenligne inputtet uafhængigt af bogstavstørrelse.

command = input("Skriv en kommando (gå nordpå/gå sydpå): ").lower()
if command == "gå nordpå":
    print("Du går nordpå.")

Game loop

I øvelse 11 har vi indkapslet koden i en while loop, så den køre indtil brugeren skriver "quit".
Det er rent faktisk en essential del af spiludvikling, at have en game loop som kører så længe spillet er aktivt.
En game loop er en løkke der kontinuerligt opdaterer spillets tilstand, håndterer bruger input, og renderer grafik.
I et tekstbaseret spil kan game loopen håndtere brugerens kommandoer og opdatere spillets tilstand baseret på disse kommandoer.

Her er et simpelt eksempel på en game loop i et tekstbaseret RPG spil:

while True:
    command = input("Skriv en kommando (gå nordpå/gå sydpå/quit): ").lower()
    if command == "quit":
        break
    elif command == "gå nordpå":
        print("Du går nordpå.")
    elif command == "gå sydpå":
        print("Du går sydpå.")
    else:
        print("Ugyldig kommando.")

Vi bruger break keyword til at afslutte game loop'en, når brugeren skriver "quit".
While True betingelse kan også skiftes ud med vores betingelse for at vinde eller tabe spillet.

Funktioner

Vi kan med fordel bruge funktioner i vores spil.
Funktioner er en måde at gruppere kode, så den kan genbruges flere gange uden at skulle skrive den samme kode igen.

def print_welcome():
    print("Velkommen til spillet!")
def print_location(location):
    print(f"Du er nu i {location}.")

Vi kan så nemt bruge funktionerne i spillet ved at kalde dem sådan her: 

print_welcome()
location = "et mørkt rum"
print_location(location)

Det er vigtigt at give funktionerne beskrivende navne, så det er nemt at forstå hvad de gør.
og at funktionerene er defineret inden de kaldes.

eksempelvis vil denne kode ikke virke: 
location = "et mørkt rum"
print_location(location)  # Dette vil give en fejl, fordi funktionen ikke er defineret endnu
def print_location(location):
    print(f"Du er nu i {location}.")

Øvelse 16: Tilret ovenstående eksempel så det virker korrekt.

Funktioner bruges ofte til at behandle data og returnere værdier efterfølgende.

def calculate_damage(attack, defense):
    damage = attack - defense
    return max(damage, 0)  # Sørger for at skaden ikke er negativ

Lad os kigge lidt på denne funktion:
Når vi bruger Return fortæller vi programmet hvad det skal returnere tilbage til det sted hvor funktionen blev kaldt.
I det her tilfælde returnerer funktionen værdien af damage, som er resultatet af at trække defense fra attack.
Hvis damage er negativt, returneres 0 i stedet.

Vi kan udtrykke det anderledes. Hvis vores defence styrke er større end attack styrken vil vi ikke få skade, og funktionen returnerer 0.
Hvis vores defence styrke er mindre end attack styrken vil vi få skade, og funktionen returnerer forskellen mellem attack og defence.

Lister

Lister er en måde at gemme flere værdier på i en enkelt variabel.
Lister kan indeholde forskellige typer af data, og de kan ændres efter oprettelsen.

my_list = [1, 2, 3, 4, 5]
print(my_list[0])  # Udskriver første element i listen (1)
my_list.append(6)  # Tilføjer et nyt element til listen
print(my_list)     # Udskriver hele listen

Man kan også fjerne elementer fra en liste ved at bruge pop() eller remove().
man kan slette en fra en liste med del my_list[0] eller del my_list
man kan også tømme en liste med my_list.clear()
eksempel:

my_list = [1, 2, 3, 4, 5]
my_list.pop()        # Fjerner det sidste element (5) fra listen
my_list.remove(2)   # Fjerner elementet med værdien 2 fra listen
del my_list[0]      # Fjerner det første element (1) fra listen
print(my_list)      # Udskriver resten af listen
