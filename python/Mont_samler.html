Mønt Samler

I dette lille spil skal ræven samle mønter på tid.
For hver mønt der samle får man point og det gælder om få flest mulige point inden tiden er gået.
Bemærk linje nummer fortæller hvor de små bider af koden skal indsættes i forhold til hinanden.
Vi skriver koden i Mu Editor og mode er sat til Pygame Zero.

Load spillets Actor's

En actor et indbygget element i Pygame Zero
Vi kan tilføre grafik med vores asset (sprite)

en actor har indbygget metoder vi kan bruge i spillet
Den første metode er .pos hvor vi sætter en start position for hver actor.

fox = Actor("fox")
fox.pos = 100,100

coin = Actor("coin")

Spil logik

Vi skal have noget bevægelse i spillet. Men først skal vi have lavet noget forberedelse.
Vi starter med at oprette tre funktioner som vi skal bruge senere.

def place_coin():
    pass

def time_up():
    pass

def update():
    pass

Vi starter med at give funktionerne en ”pass” instruktion.
Det betyder bare at vi helt bevist har valgt ikke at lave noget ”rigtig” kode.

Husk at slette ”pass” så snart vi begynder at tilføje kode til funktionen!

Random funktion

Den metode skal importeres fra et bibliotek der hedder random.

from random import randint

Forklaring

Import af bibliotek (libraries)

Ofte skal man bruge elementer i sit program som allerede findes,
man behøver derfor ikke lave dem selv.

for at få adgang til disse elementer skal man enten importer helt library eller del af et library.

Ovenfor importerede vi kun randint fra biblioteket random.
ville vi importer hele biblioteket ville koden have set sådan her ud

import random

Men vi ville så også fortælle i vores kode at vi nu vil bruge randint fra random

def place_coin():
    coin.x = random.randint(20,(WIDTH - 20))
    coin.y = random.randint(20,(HEIGHT - 20))

Skal man bruge mange funktioner for et library kan det være smart at importer det hele.
men ellers tager man kun de enkelte funktioner.

Kald place_coin()

Vi vil nu tilføje kald af funktionen så snart programet køre.
Derved får vi en tilfældig placering af mønten ved hver opstart.

place_coin()

Prøv nu at starter og stoppe spillet et par gange. Hold øje med hvor mønten er på skærmen.

Stop spillet på tid

Ændre time_up() funktionen så den ser sådan her ud.

def time_up():
    global game_over
    game_over = True

Denne linje kode placeres lige før linjen med place_coin()

clock.schedule(time_up, 7.0)

Den linje kode starter en timer på 7 sekunder så snart spillet starter.
Når tiden er gået, går programmet til funktionen time_up()
time_up() sætter vores game_over variable til True.

Nu skal vi tilføje koden som får spillet til at stoppe.
tilføj i draw() funktionen denne if betingelsen.

    if game_over == True:
        screen.fill("pink")
        screen.draw.text("Final Score:" + str(score), color="black", topleft=(10,10), fontsize=60)

Husk indrykkene skal passe!

Når skærmen gentegner vil den hvis game_over variablen er True
sætte baggrundsfarven til pink og ændre score teksten.

Kollision

Vi skal finde ud om ræven rammer mønten.
Hvis den rammer mønten skal vi øge vores score
og flytte mønten til en ny placering.

Det første er at hente score variablen ind i update, så vi kan skrive variablen

    global score

Derefter bruger vi actor (ræven) elements indbygget funktion til at se om den rammer en anden actor (mønten).

    if fox.colliderect(coin):
        score = score + 10
        place_coin()

Afprøv

Vores spil er nu færdig
Du kan nu styre ræven med piletasterne ind i mønterne.
Fanger du mønterne får du point.
Når der er gået 7 sekunder er spillet slut.
Det gælder om at få fange så mange mønter som muligt.

Komplet kode

Hele koden ser sådan ud:

from random import randint

WIDTH = 400
HEIGHT = 400
TITLE = "Coin Collector" 

score = 0
game_over = False

fox = Actor("fox")
fox.pos = 100,100

coin = Actor("coin")
coin.pos = 200,200

def draw():
    screen.fill("green")
    fox.draw()
    coin.draw()
    screen.draw.text("Score:" + str(score), color="black", topleft=(10,10))

    if game_over == True:
        screen.fill("pink")
        screen.draw.text("Final Score:" + str(score), color="black", topleft=(10,10), fontsize=60)

def place_coin():
    coin.x = randint(20,(WIDTH - 20))
    coin.y = randint(20,(HEIGHT - 20))

def time_up():
    global game_over
    game_over = True

def update():
    global score
    if keyboard.left:
        fox.x = fox.x -2
        
    if keyboard.right:
        fox.x = fox.x +2
        
    if keyboard.up:
        fox.y = fox.y -2
        
    if keyboard.down:
        fox.y = fox.y +2

    if fox.colliderect(coin):
        score = score + 10
        place_coin()
        
clock.schedule(time_up, 7.0)
place_coin()

Forslag til udvidelser

Prøv at udvide spillet på egen hånd.
Her er nogen forslag du måske kan blive inspireret af.

  • Øg hvor lang tid man har til spillet
  • Ændre hastigheden på ræven
  • Ændre Actor billedet når ræven går til venstre/højre
  • Vis hvor lang tid der er tilbage i spillet (hint. update() kaldes 60 gange i sekundet)
  • Lav bonus tid actors som dukker op på tilfældig tidspunkt og forsvinder igen
  • Flyt mønten hvis den ikke er taget efter noget tid
