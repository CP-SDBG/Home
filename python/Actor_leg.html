Actor reference pukter

Når vi ikke definer en placering af vores actor objekt
så bliver det "smidt" med top venstre hjørne på koordinat 0,0

Vi vil nu se på forskellige muligheder for at placer vores Actor efter vores ønske.

Et actor objekt har en form for usynlig boks omkring sin grafik.
Den boks har nogle forud defineret punkter vi kan benytte.

Placering af Actor 1

Vi kan definer placering samtidig med vi opretter vores actor objekt.
Prøv disse forskellige muligheder 

player = Actor("brick_face", center=(100,100))
player = Actor("brick_face", midright=(WIDTH,300))
player = Actor("brick_face", bottomleft=(0,HEIGHT))

Placering af Actor 2

Vi kan også definer placering særskilt efter vi har oprettet vores object
Prøv disse forskellige muligheder 

player = Actor("brick_face")
player.center = 200,200
  
player = Actor("brick_face")
player.midleft = 100,300

Man kan faktisk også bare bruge top, bottom, left og right sådan her

player = Actor("brick_face")
player.right = WIDTH
  
player = Actor("brick_face")
player.right = 300
player.bottom = HEIGHT

Placering af Actor 3

Et actor objekt har også en indbygget pos attribute. Med den sætter man placering for Actor center punkt. 

player = Actor("brick_face")
player.center = 100,100
  
player = Actor("brick_face")
player.pos = WIDTH/2,HEIGHT/2

Typisk vil man dog bruge .pos når man skal aflæse hvor en actor er placeret på skærmen.
Da .pos er en tuple (flere værdier i en attribute) kan man udlæse x værdien med .pos[0] og y værdien med .pos[1]

Flytte Actor

Vil man tilføje en ændring til den aktuelle placering, kan man med fordel sætte actors x og y værdier særskilt.

Prøve dette lidt større eksempel. 

WIDTH = 400
HEIGHT = 400
TITLE = "Actor Leg"

ground_height = 50

player = Actor("brick_face")
player.midbottom = WIDTH/2,(HEIGHT-ground_height)

def update():
    if keyboard.left:
        player.x -= 1
    
    if keyboard.right:
        player.x += 1
        
    if keyboard.space:
        player.y -= 10
    
    if player.y <= HEIGHT-ground_height-player.size[1]/2:
        player.y += 2
        
def draw():
    screen.clear()
    screen.fill("lightblue")
    ground()
    player.draw()
    coordinates()

def ground():
    ground = Rect((0,HEIGHT-ground_height),(WIDTH,HEIGHT))
    screen.draw.filled_rect(ground, color="lightgreen")

def coordinates():
    screen.draw.text("X: " + str(player.pos[0]) + "     Y: " + str(player.pos[1]), (10,10))
